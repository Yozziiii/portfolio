Pour installer le projet en local vous devez taper les commandes si dessous.

Cloner le projet :
git clone https://github.com/Yozziiii/portfolio.git

Allez à : 
cd portfolio

Installer y les dépendances avec : 
npm install

Lancer le projet : 
npm start

Version optimisée pour la production : 
npm run build
